# Installation

1. Install dependency

   ```bash
   npm run install
   ````

2. Run migrations

   ```bash
   npm run migration:run
   ```

3. Run seeds

   ```bash

   npm run seed

4. Run app in dev mode

   ```bash
   npm run start:dev
   ```

5. Open <http://localhost:3000>

- Swagger (API docs): <http://localhost:3000/docs>
